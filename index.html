Step 1 — Get the Google Sheets template
→ Open the Chartd Trading Journal Template
File → Make a copy → save to your Google Drive.

Step 2 — Add the Apps Script
Inside your copy: Extensions → Apps Script → delete existing code → paste this:
function doPost(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet()
    .getSheetByName("Trade Database");
  const d = JSON.parse(e.postData.contents);
  if (d.action === "appendRow") sheet.appendRow(d.row);
  return ContentService.createTextOutput("ok");
}
function doGet(e) {
  return ContentService.createTextOutput("alive");
} Step 3 — Deploy as Web App
Deploy → New Deployment → Web App
Execute as: Me  |  Who has access: Anyone
